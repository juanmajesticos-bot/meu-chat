# 💬 Meu Chat

Um chat simples, bonito e funcional feito com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- 💾 Salva mensagens no localStorage
- 👤 Sistema de nome de usuário
- 🎨 Design moderno com gradientes
- 📱 Responsivo (funciona no celular)
- ⌨️ Suporte a tecla Enter
- 🖱️ Interface intuitiva

## 🚀 Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no seu navegador
3. Digite seu nome e comece a conversar!

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)

## 📦 Estrutura

- `index.html` - Estrutura da página
- `style.css` - Estilos e responsividade
- `script.js` - Lógica do chat
